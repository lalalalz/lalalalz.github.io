# 마크다운이란?

마크다운은 존 그루버와 아론 스워츠가 이메일 글쓰기 형식에 영감을 받아 Python을 이용한 html
변환기를 만들면서 시작되었다.


즉, 마크다운은 *텍스트를 HTML로 변환*하는 언어이자 도구이다. 


# 마크다운 주요 문법

## 1. 헤더

    #      -> <h1> (밑줄 포함)
    ##     -> <h2> (밑줄 포함)
    ###    -> <h3>
    ####   -> <h4>
    #####  -> <h5>
    ###### -> <h6>

![img_마크다운_헤더](../assets/images/header.png)
    
## 2. 제목

    제목은 대제목과 소제목으로 나눌 수 있는데 이때 각각은 #, ##과 같다.
